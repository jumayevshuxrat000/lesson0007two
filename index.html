Learn JavaScript Conditionals

Master JavaScript with examples on if, else, switch statements and more.

If Statement

The if statement allows you to run code based on a condition.

if (condition) { /* code to execute */ }

Else Statement

The else statement lets you run code if the condition is false.

if (condition) { /* code */ } else { /* code */ }

Else If Statement

Use else if for multiple conditions.

if (condition1) { /* code */ } else if (condition2) { /* code */ }

Switch Statement

The switch statement evaluates an expression and matches it against multiple cases.

switch(expression) {
case value1:
// code
break;
case value2:
// code
break;
default:
// code
}

Ternary Operator

The ternary operator provides a shorthand way to write conditional statements.

condition ? /* if true */ : /* if false */

Logical Operators

Logical operators like AND (&&), OR (||), and NOT (!) are used to combine multiple conditions.

if (condition1 && condition2) { /* code */ }
if (condition1 || condition2) { /* code */ }
if (!condition) { /* code */ }
